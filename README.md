# ريحانة — نظام التشغيل (Rayhana Shift)

نظام تشغيل يومي لكافيه ريحانة: إدارة الشفتات، المهام، الحضور والانصراف، الاستراحات، تسليم الشفتات، المهارات والتدريب، والتقييم الشهري — بواجهة عربية كاملة (RTL) مصممة للهاتف.

## البنية

- **واجهة**: ملف واحد `index.html` (بدون أطر عمل) — يعمل كتطبيق ويب تقدمي (PWA) قابل للتثبيت.
- **خادم**: Supabase (PostgreSQL) عبر دالتي RPC: `rko_staff` و `rko_admin` — كل منطق العمل داخل قاعدة البيانات بصلاحيات `SECURITY DEFINER`.
- **مزامنة دون اتصال**: عمليات التعديل تُحفظ في طابور محلي (`localStorage`) وتُرسل تلقائياً عند عودة الاتصال، مع حماية من التكرار عبر `op_id`.

## أبرز الميزات

### واجهة الموظفة
- تسجيل حضور/انصراف مع حساب التأخير والإضافي تلقائياً.
- مهام يومية تولَّد آلياً من قوالب حسب المنطقة والمرحلة (افتتاح/أثناء/إغلاق) مع مؤقّت صافٍ وإيقاف مؤقت لخدمة الضيوف.
- توثيق بالمهام: صورة، قائمة فحص، رقم، أو تأكيد موظفة ثانية.
- تحديث ضغط المنطقة ونداءات المساعدة.
- تسليم شفت موثّق باعتماد المستلمة، واختيار «الأكثر تعاوناً» بسرية.

### لوحة الإدارة
- لوحة يوم بإحصائيات فورية وتنبيهات مرتّبة بالأثر، وتحديث تلقائي كل دقيقة.
- إدارة الموظفات، المناطق، أنواع الشفتات، وقوالب المهام.
- جدولة الشفتات: إنشاء، تعديل، أوقات مخصصة، ونسخ جدول يوم كامل.
- مراجعة المهام (فردي أو اعتماد جماعي) مع عرض صور التوثيق.
- الحضور: اعتماد/رفض تعديلات «نسيت أسجل»، متابعة الاستراحات المؤجلة.
- مصفوفة المهارات والتغطية الحرجة، ومسار تدريب كامل (جديد → طُبّق → اختُبر → معتمد).
- خطط تغطية الغياب بترتيب ذكي حسب الأهلية والإرهاق.
- تقارير قابلة للتصدير CSV، سجل تدقيق، وإشارات مراجعة بلا عقوبات تلقائية.

## النشر

المشروع منشور على Netlify. أي دفع إلى فرع النشر يُحدّث الموقع.

- `?reset=1` بنهاية الرابط: يمسح الجلسة والكاش المحليين (مفيد بعد التحديثات).
- `?reset=all`: يمسح أيضاً مفتاح الاتصال المحفوظ محلياً.

## دخول الإدارة

من شاشة الدخول: خمس لمسات متتالية على الشعار ثم كلمة سر الإدارة.
